1) Typography
- <Typography variant="h6" component="h2" gutterBottom noWrap> Hello </Typography>
- gutterBottom is Margin Bottom
- styles of an element is defined by this variant prop
- if want to override the variant then use component prop but the styles of variant will still be applied
- color prop has 2 color tyoes. 1. primary 2.textPrimary, remember you can only use theme colors such primary,  secondary,error,success
- noWrap prop wraps the text
